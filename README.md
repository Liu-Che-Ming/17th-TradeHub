# TradeHub

### 複製專案

```
$ git clone https://github.com/astrocamp/17th-TradeHub (複製到本地)
$ cd git_practice (進入專案內容)
$ git pull origin main (拉取最新變更)
$ git status (確認狀態)
```

### 專案初始化

**虛擬環境:**

```
$ pip install poetry
$ poetry init -n
$ poetry shell
$ poetry install
```

**Node.js**

```
$ install node
$ npm init -y
$ npm run dev
$ make server
```
